 배포 주소

 프로젝트 소개
  - 한국 각 지역을 여행할 수 있는 상품을 유저들은 장바구니에 넣거나, 검색을 할수있다.
  
 구현 기능
  - 전체 페이지 디자인, 기획, 제작 (ㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇ)
  - 항공api 를 활용하여 데이터 추출
  - Firebase database를 활용한 로그인 구현
  - Firebase database에 유저들 데이터 저장하는 장바구니 구현
  - 유저를 위한 고객센터 구현
  - 여행 후기 페이지 구현
  - 각 지역을 검색할수 있는 검색창 구현
